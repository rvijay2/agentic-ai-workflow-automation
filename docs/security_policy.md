# Security Policy

## Access Control
All employees must use multi-factor authentication (MFA) for all systems.
Passwords must be at least 16 characters and changed every 90 days.

## Data Classification
- Public: Can be shared externally
- Internal: Company employees only
- Confidential: Need-to-know basis
- Restricted: Executive and legal only

## Incident Response
1. Detect and report security incidents within 1 hour of discovery
2. Notify Security Team immediately via security@company.com
3. Preserve all logs and evidence
4. Do not attempt to remediate without Security Team approval

## Remote Work Policy
- Always use company VPN when accessing internal systems
- Never use public WiFi without VPN
- Lock workstation when unattended
- Use encrypted storage for all work data
